[Research Project] Sign langugage sensor glove and education system

This project aims to improve upon prior sign language recognition glove by using inertial measurement units (IMU) to detect gesture in space. Along with the improved wearable, an education application will use the data from the glove and 2D camera to teach and instruct user of correct sign.

Website
[Research Project] Project Sensory Ultrasonic Navigation

Project S.U.N. is an engineering project which utilizes ultrasonic sensors to aid the visually impaired in navigation. Two sets of wearables, a glove and shoulder component, were created to house the ultrasonic sensors and vibrational and piezo motors that gave tactile and auditory feedback to the user. This project won "Best in Fair" in a regional science fair and went on to participate in the Intel International Science and Engineering Fair (ISEF).



Teammates: Zachary Guo & Jeremy Tan
